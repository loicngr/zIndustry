### Simple canvas game write in TypeScript. [Lit](https://lit.dev/) is used for UI and [Hilla Router](https://hilla.dev/docs/routing/router) for router.

### Inspiration

- [w3Schools](https://www.w3schools.com/graphics/game_intro.asp)

### Commands

- npm run dev
- npm run build
